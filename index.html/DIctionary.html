бһҹбһ бһӮбһҳбһ“бҹҚбҹ” бһӣбҹ„бһҖбһ”бһ¶бһ“бһ…бһјбһӣбһҡбһҪбһҳбһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бһ¶бһҖбҹӢбһ‘бһ„ бһӣбҹ„бһҖ бһҹбҹҒбһ„ бһҹбҹҠбһёбһҗбһ¶ / SENG SITHA бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһҮбһ¶бһӮбҹ’бһҡбһјвҖӢбһ”бһ„бҹ’бһҡбҹҖбһ“ бһ“бһ·бһ„ бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҖбһ¶бһҡбһңбһ·бһ‘бҹ’бһҷбһ¶вҖӢвҖӢбһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҹбһ бһӮбһҳбһ“бҹҚ (ICD)бҹ” бһӣбҹ„бһҖбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бһҡбһ·бһүбҹ’бһүбһ¶бһ”бһҸбҹ’бһҡбһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢ (Master's Degree) бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ” бһӣбҹ„бһҖбһ”бһ¶бһ“бһ…бһјбһӣбһҡбһҪбһҳбһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶвҖӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбһҳбһ“бҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҠбһјбһ…бһҮбһ¶ бһ”бһ¶бһ“бһ”бҹ’бһҡбһЎбһ„бһҮбһ¶бһ”бҹӢбһӮбҹ’бһҡбһјбһҖбһҳбҹ’бһҡбһ·бһҸбһ§бһҸбҹ’бһҸбһҳбҹ” бһӣбҹ„бһҖбһ”бһ¶бһ“бһ…бһјбһӣбһ”бһҳбҹ’бһҡбһҫбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ…бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҹбһ бһӮбһҳбһ“бҹҚбһ“бҹҒбҹҮбһҸбһ¶бҹҶбһ„бһ–бһёбһҗбҹ’бһ„бҹғбһ‘бһё вҖӢбҹ бҹЎ бһңбһ·бһ…бҹ’бһҶбһ·бһҖбһ¶ бҹўбҹ бҹўбҹЈ бһ бһҫбһҷбһҳбһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ”

бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ 10+ бһҶбҹ’бһ“бһ¶бҹҶ бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷ IT

бһҮбҹҶбһ“бһ¶бһүбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’

бһ”бһ„бҹ’бһҡбҹҖбһ“ бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ“бһ·бһҹбҹ’бһҹбһ·бһҸ

YIM SOKKHOEUN

бһӣбҹ„бһҖ бһҷбһ№бһҳ бһҹбһ»бһҒбһғбһҝбһ“ / YIM SOKKHOEUN бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһҮбһ¶бһӮбҹ’бһҡбһјвҖӢбһ”бһ„бҹ’бһҡбҹҖбһ“ бһ“бһ·бһ„ бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҖбһ¶бһҡбһңбһ·бһ‘бҹ’бһҷбһ¶вҖӢвҖӢбһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҹбһ бһӮбһҳбһ“бҹҚ (ICD)бҹ” бһӣбҹ„бһҖбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бһҡбһ·бһүбҹ’бһүбһ¶бһ”бһҸбҹ’бһҡ (Master's Degree) бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ” бһӣбҹ„бһҖбһ”бһ¶бһ“бһ…бһјбһӣбһҡбһҪбһҳбһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶвҖӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбһҳбһ“бҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҠбһјбһ…бһҮбһ¶ бһ”бһ¶бһ“бһ”бҹ’бһҡбһҖбһ¶бһҹбһ”бһүбҹ’бһ…бҖёбһӣбһҖбҹ’бһҒбһҺбҖ®бҹ” бһӣбҹ„бһҖбһ”бһ¶бһ“бһ…бһјбһӣбһ”бһҳбҹ’бһҡбһҫбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ…бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҹбһ бһӮбһҳбһ“бҹҚбһ“бҹҒбҹҮбһҸбһ¶бҹҶбһ„бһ–бһёбһҗбҹ’бһ„бҹғбһ‘бһё вҖӢбҹ бҹЎ бһҖбһүбҹ’бһүбһ¶ бҹўбҹ бҹЎбҹЁ бһ бһҫбһҷбһҳбһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ”

бһҮбҹҶбһ“бһ¶бһүбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’

NHOUNG SAMNANG

бһӣбҹ„бһҖ бһүбһјбһ„ бһҹбҹҶбһҺбһ¶бһ„ / NHOUNG SAMNANG бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһҮбһ¶бһӮбҹ’бһҡбһјвҖӢбһ”бһ„бҹ’бһҡбҹҖбһ“ бһ“бһ·бһ„ бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҖбһ¶бһҡбһңбһ·бһ‘бҹ’бһҷбһ¶вҖӢвҖӢбһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҹбһ бһӮбһҳбһ“бҹҚ (ICD)бҹ” бһӣбҹ„бһҖбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бһҡбһ·бһүбҹ’бһүбһ¶бһ”бһҸбҹ’бһҡ (Master's Degree) бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ” бһӣбҹ„бһҖбһ”бһ¶бһ“бһ…бһјбһӣбһҡбһҪбһҳбһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶвҖӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбһҳбһ“бҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҠбһјбһ…бһҮбһ¶ бһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһӮбһҡбһ»бһҖбҹ„бһҹбһӣбҹ’бһҷ (бһ”бһҡбһ·бһүбҹ’бһүбһ¶бһ”бһҸбҹ’бһҡ+бҹЎ)бҹ” бһӣбҹ„бһҖбһ”бһ¶бһ“бһ…бһјбһӣбһ”бһҳбҹ’бһҡбһҫбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ…бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҹбһ бһӮбһҳбһ“бҹҚбһ“бҹҒбҹҮбһҸбһ¶бҹҶбһ„бһ–бһёбһҗбҹ’бһ„бҹғбһ‘бһё вҖӢбҹ бҹЎ бһҸбһ»бһӣбһ¶ бҹўбҹ бҹўбҹў бһ бһҫбһҷбһҳбһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ”

LOEURN SINA

бһҖбһүбҹ’бһүбһ¶вҖӢ бһӣбһҝбһ“ бһҹбҹҠбһёбһҺбһ¶ / LOEURN SINA бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһҮбһ¶бһӮбҹ’бһҡбһјвҖӢбһ”бһ„бҹ’бһҡбҹҖбһ“ бһ“бһ·бһ„ бһ”бҹ’бһҡбһ…бһ¶бҹҶбһҖбһ¶бһҡбһңбһ·бһ‘бҹ’бһҷбһ¶вҖӢвҖӢбһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҹбһ бһӮбһҳбһ“бҹҚ (ICD)бҹ” бһҖбһүбҹ’бһүбһ¶бһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бһҡбһ·бһүбҹ’бһүбһ¶бһ”бһҸбҹ’бһҡ (Master's Degree) бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ” бһҖбһүбҹ’бһүбһ¶бһ”бһ¶бһ“бһ…бһјбһӣбһҡбһҪбһҳбһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶвҖӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбһҳбһ“бҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҠбһјбһ…бһҮбһ¶ бһ”бһ¶бһ“бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһӮбһҡбһ»бһҖбҹ„бһҹбһӣбҹ’бһҷ (бһ”бһҡбһ·бһүбҹ’бһүбһ¶бһ”бһҸбҹ’бһҡ+бҹЎ)бҹ” бһҖбһүбҹ’бһүбһ¶бһ”бһ¶бһ“бһ…бһјбһӣбһ”бһҳбҹ’бһҡбһҫбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ…бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҹбһ бһӮбһҳбһ“бҹҚбһ“бҹҒбҹҮбһҸбһ¶бҹҶбһ„бһ–бһёбһҗбҹ’бһ„бҹғбһ‘бһё вҖӢбҹ бҹЎ бһҸбһ»бһӣбһ¶ бҹўбҹ бҹўбҹў бһ бһҫбһҷбһҳбһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ”

бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ 10+ бһҶбҹ’бһ“бһ¶бҹҶ бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷ IT

бһҮбҹҶбһ“бһ¶бһүбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’

бһ”бһ„бҹ’бһҡбҹҖбһ“ бһ“бһ·бһ„бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһ“бһ·бһҹбҹ’бһҹбһ·бһҸ

HENG CHEA

бһӣбҹ„бһҖ бһ бҹҒбһ„ бһҮбһ¶ / HENG CHEA бһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһҮбһ¶бһӮбҹ’бһҡбһјвҖӢбһ”бһ„бҹ’бһҡбҹҖбһ“бһҹбһ»бһ‘бҹ’бһ’бһңбһ·бһ‘бҹ’бһҷбһ¶вҖӢвҖӢбһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҹбһ бһӮбһҳбһ“бҹҚ (ICD)бҹ” бһӣбҹ„бһҖбһҳбһ¶бһ“бһҖбһҳбҹ’бһҡбһ·бһҸбһҹбһ·бһҖбҹ’бһҹбһ¶бһ”бһҡбһ·бһүбҹ’бһүбһ¶бһ”бһҸбҹ’бһҡ (Master's Degree) бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ” бһӣбҹ„бһҖбһ”бһ¶бһ“бһ…бһјбһӣбһҡбһҪбһҳбһңбһӮбҹ’бһӮбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһҹбһ·бһҖбҹ’бһҒбһ¶бһҹбһ¶бһӣбһ¶бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһўбһ”бҹӢбһҡбҹҶвҖӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ бһӮбһҳбһ“бҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҡбһ”бһҹбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҠбһјбһ…бһҮбһ¶ бһ”бһ¶бһ“бһ”бҹ’бһҡбһҖбһ¶бһҹбһ”бһүбҹ’бһ…бһјбһӣбһҖбҹ’бһҡбһ”бһҒбҹҗбһҺбҹ’бһҢбҹ” бһӣбҹ„бһҖбһ”бһ¶бһ“бһ…бһјбһӣбһ”бһҳбҹ’бһҡбһҫбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡбһ“бҹ…бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбҹ’бһҗбһ¶бһ“бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҹбһ бһӮбһҳбһ“бҹҚбһ“бҹҒбҹҮбһҸбһ¶бҹҶбһ„бһ–бһёбһҗбҹ’бһ„бҹғбһ‘бһё вҖӢбҹ бҹЎ бһҖбһүбҹ’бһүбһ¶ бҹўбҹ бҹЎбҹЁ бһ бһҫбһҷбһҳбһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһўбһ”бҹӢбһҡбҹҶ бһ“бһ·бһ„бһҹбһ бһӮбһҳбһ“бҹҚбҹ”

бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ 10+ бһҶбҹ’бһ“бһ¶бҹҶ бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷ IT
